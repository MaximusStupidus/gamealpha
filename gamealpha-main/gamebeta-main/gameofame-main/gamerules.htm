Game Rules

1. This is an interactive game in which the player is given a grid styled picture with different boxes colored to make the famous personality unrecognizable.

2. The player will be given hints at various stages of the game.

3. If the player is not able to guess the personality from hint 1, they can select the option to go to hint 2. This will reduce the points to be awarded but make the picture a little bit clearer by removing the color from the boxes for that respective hint.

4. Ensure the highest score to win with the least possible hints.
